Simple weight based shipping method for WooCommerce

== Description ==

Plugin adds a simple weight based shipping method in WooCommerce. You can find its settings under the WooCommerce -> Settings -> Shipping.

Plugin is moved to http://codecanyon.net/. Plese look for it there.

== Installation ==

1. Upload `woocommerce-weight-based-shipping` folder  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
